# Historical Gold Price Analysis (1833 - 2026)

## Project Overview
This project analyzes nearly 200 years of gold price data to identify patterns in market volatility and annual returns. The main goal was to compare absolute price growth with relative year-over-year (YoY) performance.

## Key Insights
* **Volatility:** While current prices are at an all-time high, the 1970s represented the most volatile period in history with the highest relative returns.
* **Methodology:** Used Python (Pandas) for data cleaning and Matplotlib for visualizing long-term trends versus percentage changes.

## Tech Stack
* **Language:** Python 3
* **Libraries:** Pandas, NumPy, Matplotlib
* **Environment:** Kaggle / Jupyter Notebook
